1 Propositional Logic

We begin by defining the syntax and semantics of propositional logic, using an inductive type of formulas, valuations, and the notions of satisfiability.

1.1 Formulas

Definition 1.1
✓
#

Lean declarations

  • LHCP.Formula

The set of formulas \(\mathsf{Formula}\) is defined inductively as

  • \(\mathsf{Atom}(n) \in \mathsf{Formula}\).

  • \(\mathsf{Neg}(f) \in \mathsf{Formula}\) for all \(f \in \mathsf{Formula}\).

  • \(\mathsf{Imp}(f_1, f_2) \in \mathsf{Formula}\) for all \(f, g \in \mathsf{Formula}\).

We write \(\neg \mathsf{Formula}\) for \(\mathsf{Neg}(\mathsf{Formula})\) when the notation is nonambiguous.

We write \(f_1 \to f_2\) for \(\mathsf{Imp}(f_1, f_2)\) when the notation is nonambiguous.

1.2 Valuations and Semantics

Definition 1.2
✓
#

Lean declarations

  • LHCP.Valuation

A valuation is a function \(v : \mathbb {N}\to \mathsf{Bool}\) assigning truth values to atoms.

Definition 1.3
✓
#

Uses

  • Definition 1.1
  • Definition 1.2

Lean declarations

  • LHCP.Valuation.eval

Given a valuation \(v\) and a formula \(f\), the evaluation \(v.\mathsf{eval}(f) : \mathsf{Bool}\) is defined recursively in \(f\) as

\[ \begin{aligned} v.\mathsf{eval}(\mathsf{Atom}(n)) & := v(n), \\ v.\mathsf{eval}(\mathsf{Neg}(f_1)) & := \neg \, v.\mathsf{eval}(f_1), \\ v.\mathsf{eval}(\mathsf{Imp}(f_1, f_2)) & := \textbf{if}\ v.\mathsf{eval}(f_1) = \mathsf{true}\ \textbf{then}\ v.\mathsf{eval}(f_2)\ \textbf{else}\ \mathsf{true} \end{aligned} \]

We write \(v[f]\) for \(v.\mathsf{eval}(f)\).

Definition 1.4
✓
#

Uses

Lean declarations

  • LHCP.Satisfies

A valuation \(v\) satisfies a formula \(F\), written \(v \models F\), if \(v.\mathsf{eval}(F) = \mathsf{true}\).

Definition 1.5
✓
#

Uses

  • Definition 1.4

Lean declarations

  • LHCP.Satisfiable

A formula \(F\) is satisfiable, written \(F \ \mathsf{sat}\), if there exists a valuation \(v\) such that \(v \models F\).

1.6 Tseitin encoding

Definition 1.13
✓
#

Uses

  • Lemma 1.12

Lean declarations

  • LHCP.Tseitin.V

For each formula \(f\), \(V_f\) is the atom given by the injection in 1.12.

Definition 1.14
✓
#

Uses

  • Definition 1.13

Lean declarations

  • LHCP.Tseitin.N

For a formula \(f\), \(N(f)\) is the list of formulas

\[ N(f) := \begin{cases} [\ ] & f = \mathsf{Atom}(n) \\ \left[ V_f \leftrightarrow \neg V_g \right] & f = \neg g \\ \left[ V_f \leftrightarrow (V_g \to V_h) \right] & f = g \to h \end{cases} \]
Definition 1.15
✓
#

Uses

  • Definition 1.14

Lean declarations

  • LHCP.Tseitin.Ns

For a formula \(f\), \(N^*(f)\) is the list of all \(N(g)\) of all subformulas \(g\) of \(f\).

\[ N^*(f) := \begin{cases} [\ ] & f = \mathsf{Atom}(n) \\ N(f) \cup N^*(g) & f = \neg g \\ N(f) \cup N^*(f_1) \cup N^*(f_2) & f = f_1 \to f_2 \end{cases} \]

where \(\cup \) denotes list concatenation.

Definition 1.16
✓
#

Uses

  • Definition 1.15
  • Definition 1.8
  • Definition 1.9

Lean declarations

  • LHCP.Tseitin.T

Let \(f\) be a non-atomic formula. Tseitin’s T of \(f\) is the conjunction of all \(N(g)\) of all subformulas \(g\) of \(f\). Equivalently, it’s the conjunction of \(N^*(f)\).

\[ T(f) := \bigwedge N^*(f) \]
Definition 1.17
✓
#

Uses

  • Definition 1.16
  • Definition 1.13

Lean declarations

  • LHCP.Tseitin.E

Let \(f\) be a formula. The Tseitin encoding \(E(f)\) is \(T(f) \land V_f\) unless \(f\) is atomic, in which case it’s just \(V_f\).

\[ E(f) := \begin{cases} V_f & f = \mathsf{Atom}(n) \\ T(f) \wedge V_f & \text{otherwise}\\ \end{cases} \]

1.7 Atoms and subformulas

Definition 1.18
✓
#

Uses

  • Definition 1.1

Lean declarations

  • LHCP.Tseitin.Atoms

For a formula \(f\), define \(\mathsf{Atoms}(f)\) to be the (finite) set of indices of all atomic subformulas of \(f\). It is defined recursively as:

\[ \begin{aligned} \mathsf{Atoms}(\mathsf{Atom}(n)) & := \{ n\} \\ \mathsf{Atoms}(\neg g) & := \mathsf{Atoms}(g) \\ \mathsf{Atoms}(g \to h) & := \mathsf{Atoms}(g) \cup \mathsf{Atoms}(h) \\ \end{aligned} \]
Definition 1.19
✓
#

Uses

  • Definition 1.1

Lean declarations

  • LHCP.Tseitin.Sub

For a formula \(f\), define \(\mathsf{Sub}(f)\) to be the (finite) set of subformulas of \(f\). It is defined recursively as:

\[ \begin{align} \mathsf{Sub}(\mathsf{Atom}(n)) & := \{ \mathsf{Atom}(n)\} \\ \mathsf{Sub}(\neg g) & := \{ \neg g\} \cup \mathsf{Sub}(g) \\ \mathsf{Sub}(g \to h) & := \{ g \to h\} \cup \mathsf{Sub}(g) \cup \mathsf{Sub}(h) \\ \end{align} \]

1.11 Tseitin Equivalence

Lemma 1.38
✓
#

Uses

  • Lemma 1.25
  • Lemma 1.26

Lean declarations

  • eval_not

If \(f\) is non-atomic and \(w \models T(f)\), then for every \(g\) such that \(\neg g\) is a subformula of \(f\), we have

\[ w(V_{\neg g}) = \neg \, w(V_g). \]
Proof ▶

The clause \(V_{\neg g} \leftrightarrow \neg V_g\) is in \(N(\neg g)\), hence it is in \(N^*(f)\). Since \(w \models T(f)\), \(w \models g\), for all clauses \(g\) in the conjuction \(T(f) = \bigwedge N^*(f)\). Therefore \(w \models V_{\neg g} \leftrightarrow \neg V_g\) which by 1.25 is the equality we are looking for.

Lemma 1.39
✓
#

Uses

  • Lemma 1.25
  • Lemma 1.26

Lean declarations

  • eval_imp

If \(f\) is non-atomic and \(w \models T(f)\), then for every \(g_1, g_2\) such that \((g_1 \to g_2)\) is a subformula of \(f\), we have

\[ w(V_{(g_1 \to g_2)}) = (\neg w(V_{g_1}) \; \lor \; w(V_{g_2})). \]
Proof ▶

As with 1.38, the clause \(V_{(g_1 \to g_2)} \leftrightarrow (V_{g_1} \to V_{g_2})\) lies in \(N^*(f)\), and is therefore satisfied by \(w\).

Theorem 1.40
✓
#

Uses

  • Definition 1.18
  • Lemma 1.20
  • Lemma 1.36
  • Lemma 1.37
  • Lemma 1.38
  • Lemma 1.39

Lean declarations

  • eval_eq_of_mem_ssub

Let \(f\) be non-atomic such that \(w \models T(f)\). If \(v\) and \(w\) agree on all atomic variables of \(f\), then for every proper subformula \(g\) of \(f\) we have

\[ w(V_g) = v[g]. \]
Proof ▶

By induction on \(g\).

  • Case \(g=\mathsf{Atom}(n)\): follows from hypothesis that \(v\) and \(w\) agree on atoms.

  • Case \(g=\neg h\): use 1.38 and the induction hypothesis on \(h\) with 1.36.

  • Case \(g=(g_1 \to g_2)\): use 1.39 and the induction hypotheses on \(g_1,g_2\) with 1.37.
